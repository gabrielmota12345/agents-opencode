# Architecture Decision Records (ADRs)

Um **ADR** documenta uma decisão arquitetural significativa: o contexto, as
opções consideradas, a decisão tomada e as consequências.

## Formato

Cada ADR é um arquivo `ADR-NNNN-<titulo>.md` seguindo o template em
[`template.md`](template.md).
